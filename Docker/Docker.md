# 0. What is Docker

Docker是一种容器技术(A tool for creating and managing containers)

容器是一个标准的软件单元，也是代码与其依赖的集合。它保证了代码运行的一致性（不用配环境了，也不用理清多如牛毛的版本关系）



Docker拥有一个轻量级的操作系统以及一个Docker Engine，这使得它比虚拟机拥有更好的性能
